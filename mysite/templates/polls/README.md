### CSRF_TOKEN tag
When we use FORM(POST Method) in Django Template,
for preventing CSRF we have to add {% csrf_tokken %} tag.
(https://docs.djangoproject.com/en/2.2/ref/templates/builtins/#std:templatetag-csrf_token)
